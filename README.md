##  Installation

Follow these steps to set up the project locally.

### 1. Clone the repository
```bash
git clone https://github.com/DYNAMAXD/ExplainabilityOnBrainTumorDataset.git
cd ExplainabilityOnBrainTumorDataset
```

### 2. Create a Virtual Environment
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux / Mac
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Run the Backend (FastAPI)
```bash
python app.py
```

### 5. Frontend Development
Open the `vercel deployment` folder and serve the `index.html` file using a live server.
